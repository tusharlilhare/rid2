Introduction to HTML


❖ What is HTML?


➢ HTML stands for Hyper Text Markup Language.

➢ HTML is the standard markup language for creating Web pages.

➢ HTML elements tell the browser how to display the content.

➢ HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

➢ Technically, HTML is a markup language rather than a programming language.

➢ HTML is not case-sensitive.


❖ Hyper Text:


➢ Hyper Text means "Text within Text." A text with a link within it is hypertext.


❖ Markup Language:


➢ A markup language is a computer language used to apply layout and formatting conventions to a text document. It makes the text more interactive and dynamic.


❖ Developer:


➢ HTML was developed by Sir Tim Berners-Lee, a British computer scientist, in 1990 while working at CERN (the European Organization for Nuclear Research). He is often credited as the inventor of the World Wide Web. The Father of HTML is known as Sir Tim Berners-Lee.


❖ A Simple HTML Document:



Example:


<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <h1>This is a Heading</h1>
        <p>This is a paragraph.</p>
    </body>
</html>

❖ Explanation of Elements:


  • <!DOCTYPE html>: Declares the document type (HTML5 in this case).
  • <html>: Defines the language of the document.
  • <head>: Contains metadata like character set, responsive viewport settings, description, and author.
  • <header>: Includes the title or banner of the page and a navigation menu with links.
  • <main>: Contains the main content of the page divided into sections (<section>).
  • <section>: Used to group related content, such as "About Us" or "Our Services."
  • <footer>: Provides contact information and copyright details.

❖ What is an HTML Element?


➢ An HTML element is the basic building block of HTML (HyperText Markup Language), used to structure content on the web. An HTML element typically consists of:


1. Opening Tag:


Defines the start of the element and includes the element's name. For example, <p> indicates a paragraph element.


Content:


The information or nested elements inside the opening and closing tags.


Closing Tag:


Defines the end of the element, written with a forward slash before the element name, like </p>.


Attributes (optional)


Provide additional information or settings for the element, written inside the opening tag. For example: <a href="https://example.com">.

Example:


<p>This is a paragraph.</p>
  • Opening Tag: <p>
  • Content: This is a paragraph.
  • Closing Tag: </p>
