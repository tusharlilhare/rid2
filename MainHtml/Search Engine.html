Search Engine


Definition


A search engine is a software system designed to perform web searches systematically. It is essentially a set of programs that retrieves specific information based on a user's textual query.

Use: Search engines are used to explore the World Wide Web (WWW) in a structured manner, retrieving information relevant to the specified query.

Conventional Search: Traditional library catalog systems allow searching by:

  • Keyword
  • Title
  • Author

Types of Search Engines


  • Conventional - Based on traditional library-style keyword matching.
  • Text-Based - Processes and retrieves results based on textual queries.
  • >Voice-Based - Responds to voice commands (e.g., Siri, Google Assistant).
  • Multimedia Search - Searches for images, videos, and other non-text content.
  • Q/A Systems - Provides direct answers to questions (e.g., Wolfram Alpha).
  • Clustering Search - Groups search results into categories.
  • Research Systems - Specialized systems for academic or professional research.

History


Birth of Search Engines: The first search engine, "Archie", was created on September 10, 1990 by Alan Emtage. (Note: "Archie" was not based on the indexing concept).

Yahoo!: The first widely popular search engine, founded in 1994 by Jerry Yang and David Filo.

Google: Launched in 1998 by Larry Page and Sergey Brin, introducing the PageRank algorithm, indexing, and hyperlink-based search.

Types of Search Engines


  1. Conventional Search Engines
    • Based on traditional methods like library catalog systems.
    • Users search using keywords, titles, or authors to locate information.
  2. Text-Based Search Engines
    • Operate on textual queries entered by users.
    • Provide results matching the text input in websites, documents, or databases.
    • Example: Google, Bing.
  3. Voice-Based Search Engines
    • Allow users to perform searches using voice commands instead of typing.
    • Use speech recognition technology to process queries.
    • Example: Google Assistant, Siri, Alexa.
  4. Multimedia Search Engines
    • Specialize in searching for non-textual content such as images, videos, and audio files.
    • Example: Google Images, YouTube Search, Pinterest.
  5. Question/Answer (Q/A) Systems
    • Focused on providing direct answers to user questions instead of links to resources.
    • Often use advanced algorithms and natural language processing (NLP).
    • Example: Wolfram Alpha, ChatGPT-based systems.
  6. Clustering Search Engines
    • Organize search results into clusters or categories for easier navigation.
    • Useful when dealing with large amounts of data.
    • Example: Carrot2.
  7. Research Systems
    • Tailored for academic or specialized research purposes.
    • Provide access to scholarly articles, patents, and technical papers.
    • Example: Google Scholar, Microsoft Academic.

How Search Engines Work


  1. Crawling
    • Also known as Spider or Spider Bot.
    • An internet bot systematically browses the web for collecting data.
    • Operated by search engines for the purpose of web indexing.
  2. Indexing
    • Involves collecting, parsing, and storing data from crawled websites.
  3. Ranking
    • Determines the position a website or webpage holds on a search engine results page (SERP).

Search Results


Search results refer to the output of a query entered into a web search engine. The goal is to satisfy the user's information needs effectively and efficiently.
