World Wide Web (WWW)



Definition


The World Wide Web (WWW) is a global system of interconnected documents and resources linked by hyperlinks and URLs. Known as the Web, it is an enabling technology for accessing and sharing information online.

History

  • Inventor: Tim Berners-Lee, a computer scientist at CERN (European Organization for Nuclear Research), established in 1954.
  • Invention Year: 1989.
  • The first proposal for the Web was written in 1989.
  • By the end of 1990, a working system was implemented, which included:
    • WWW Browser
    • HTTP Server

Key Concepts

HTML (Hypertext Markup Language)

  • The language used to create web pages and web applications.

Linking

  • The process of connecting web pages via hyperlinks for navigation.

WWW Prefix

  • Includes common domain suffixes such as .com, .org, .net, etc.

Scheme Specifiers

  • Protocol identifiers like http:// or https:// to define how resources are accessed.

Web Page

  • An HTML document accessible on the WWW, often containing text, images, and hyperlinks.

Website

  • A collection of interconnected web pages hosted under a common domain name.
  • Example: www.example.com.

Browser

  • A software application designed to open and display websites.
  • Examples: Google Chrome, Mozilla Firefox, Safari.

Search Engine

  • A software program/system designed to perform web searches by indexing and ranking web pages.
  • Examples: Google, Bing, Yahoo.

Server

  • A hardware or software system that accepts and responds to client requests over a network.
  • Example: Web servers like Apache or NGINX.

Cookie

  • A small piece of data sent from a website and stored on the user's device by the browser.
  • Cookies help remember user preferences and maintain sessions.
  • They are stateful, meaning they store session data for continuity.

Deep Web

  • The invisible or hidden web, consisting of web content not indexed by standard search engines.
  • Credited to Michael K. Bergman in 2001.
  • Examples: Private databases, academic journals, or subscription-based content.

Caching

  • A web cache is a server that stores recently accessed web pages to improve response time for users.
  • Helps reduce server load and speeds up content delivery.

Security

  • Protects web users, systems, and data from unauthorized access, cyberattacks, and threats.
  • Examples: HTTPS encryption, firewalls, multi-factor authentication, and anti-malware tools.

Privacy

  • Safeguards user personal data and ensures online activities remain confidential and protected.
  • Examples: Compliance with regulations like GDPR, cookie consent policies, and anonymization of user data.

Standards

  • Guidelines and protocols established to ensure consistency, interoperability, and accessibility across the web.
  • Examples: HTML, CSS, JavaScript standards, and accessibility protocols like WCAG.
