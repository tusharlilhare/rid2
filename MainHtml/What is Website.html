What is Website



Definition of a Website


A website is a collection of interconnected web pages and related content that is identified by a common domain name. Websites can be accessed through the internet via a web browser.


Types of Websites


    1. Static Website
  • Composed of a series of pre-written HTML files.
  • Each file corresponds to a physical page of the website.
  • Content is fixed and does not change dynamically.

    • 2. Dynamic Website
  • Automatically customizes or changes its content based on user interaction, preferences, or other factors.
  • Often relies on server-side scripting and databases to generate content dynamically.

Key Components of a Website


User Browser:

  • The software used by individuals to access websites (e.g., Chrome, Firefox, Safari).

Search Engine:

  • A tool (e.g., Google, Bing) that helps users find websites by indexing and ranking them based on relevance to search queries.

DNS (Domain Name System):

  • Translates domain names (e.g., www.example.com) into IP addresses so browsers can connect to the correct server.

Server:

  • A computer system that hosts the website's files and delivers them to the user's browser when requested.
