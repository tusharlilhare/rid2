HTML Document Structure

Breakdown of the Full Form:

Every HTML document has a basic structure. Here’s a template of the essential parts:



basic structure :
!DOCTYPE html:.

Declares the document type and HTML version (HTML5).

html:

The root element of an HTML page.

head:

Contains meta-information about the document (like title, charset, and links to stylesheets).

meta charset="UTF-8":

Sets the character encoding (UTF-8 is standard for web pages).

title:

Sets the page title, displayed in the browser tab.

body:

Contains all visible content on the webpage.
